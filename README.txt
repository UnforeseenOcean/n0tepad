n0tepad

lives at: http://n0tepad.com
code at: http://github.com/rottytooth/n0tepad

A version of Notepad expressly for editing images -- via text display of binary data or a mapping to the text of the Adobe Photoshop EULA

created & coded by Daniel Temkin and Jeff Donaldson

----------

To use:

This is a C# web app. Use it with Windows or Mono -- you can compile with Visual Studio 2010 Express http://www.microsoft.com/visualstudio/eng/products/visual-studio-express-products

Requires .Net Framework 4.0
